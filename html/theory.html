Що таке діофантові рівняння?

Діофантові рівняння - невизначені поліноміальні рівняння з цілими коефіцієнтами, в яких невідомі змінні можуть набувати тільки цілих значень. Названі на честь давньогрецького математика Діофанта Александрійського.

На відміну від звичайних рівняннь, діофантові рівняння мають скінченну кількість розв'язків або не мають розв'язків взагалі.

Лінійні діофантові рівняння

Лінійні діофантові рівняння - це рівняння виду a x + b y = c , де a , b , c - цілі числа, а x , y - невідомі.

Для знаходження приватних розв'язків x 0 , y 0 , для початку потрібно знайти найбільший спільний дільник d = НСД ( a , b ) .

Далі перевіряємо умову існування розв'язків, чи ділиться c на d без остачі ( c % d = 0 ) .

Якщо так, то знаходимо приватний розв'язок ( x 0 , y 0 ) за допомогою розширеного алгоритму Евкліда.

Піфагорові трійки

Велика теорема Ферма стверджує, що не існує таких трьох натуральних чисел x , y , z , які б задовольняли рівнянню x n + y n = z n для будь-якого натурального числа n > 2 .

Для n = 2 маємо піфагорові трійки, які можна відтворити маючи тільки якийсь один з атрибутів (катет a , катет b або гіпотенузу c ).

Для відтворення трійки маємо наступні формули:
a = m 2 - n 2 - якщо дано непарний катет
b = 2 m n - якщо дано парний катет
c = m 2 + n 2 - якщо дано гіпотенузу
де m і n - натуральні числа і m > n > 0 .

Для перевірки трійки перевіряємо умову: a 2 + b 2 = c 2 .

Задача про монети (число Фробеніуса)

Задача про монети полягає у знаходженні найбільшої недосяжної суми, яку не можна зібрати з заданих номіналів монет.
Математично задача зводиться до лінійної форми діофантового рівняння p x + q y = n , де p та q - номінали монет, а x та y - їхня кількість.
Сенс полягає у знаходженні найбільшого числа n , для якого це рівняння не матиме невід'ємних цілих розв'язків. Це і буде число Фробеніуса.

Щоб знайти всі недосяжні суми для двох номіналів, спочатку треба визначити НСД.
Якщо НСД = 1 , тоді найбільша недосяжна сума (число Фробеніуса) існує.
Якщо НСД > 1 , тоді найбільша недосяжна сума (число Фробеніуса) НЕ існує, а недосяжних сум існуватиме нескінченно багато, в свою чергу досяжні суми будуть кратними НСД.
Якщо номінали взаємно прості ( НСД = 1 ) , використовується формула g ( p , q ) = p q - p - q для знаходження числа Фробеніуса.
Будь-яка сума, яка є більшою за число Фробеніуса вважається досяжною.
Для знаходження кількості недосяжних сум використовується формула N ( p , q ) = ( p - 1 ) ( q - 1 ) 2 .

Представлення числа як суми двох квадратів

Задача представляє собою розв'язання діофантового рівняння вигляду x 2 + y 2 = n .

Таке рівняння має геометричну інтерпретацію. На координатній площині воно описує коло з центром у початку координат. Знаходження цілих чисел x та y геометрично означає пошук точок які знаходяться чітко на перетині ліній кола і місцях перетину цілочисельної сітки.

За теоремою Ферма-Ейлера про суму двох квадратів, натуральне число n можна подати як суму двох квадратів цілих чисел тоді, й тільки тоді, коли при розкладанні його на прості множники кожен дільник вигляду p = 4 k + 3 входить у парному степені. Якщо хоча б один такий множник має непарний степінь, то розв'язків не існує.
Натомість, прості дільники, які при діленні на 4 дають остачу 1 вигляду p = 4 k + 1 , (а також число 2), можуть входити у розклад у будь-якому степені, бо вони завжди можуть бути розкладені на суму двох квадратів.

Для знаходження всіх можливих розв'язків використовується наступний алгоритм:
Спочатку розкладається задане число n на прості множники і перевіряється виконання умови теореми Ферма-Ейлера.
Далі кожен унікальний множник подається у вигляді суми квадратів, де:

  • Число 2 записується як 1 2 + 1 2 .
  • Прості множники вигляду p = 4 k + 1 розкладаються на базові квадрати a 2 + b 2 .
  • Прості множники вигляду p = 4 k + 3 (які обов'язково стоять у парному степені 2 m ) записуються з нулем: ( p m ) 2 + 0 2 .

І нарешті, для отримання фінального роз'язку для числа n , розв'язки його множників послідовно перемножуються за допомогою тотожності Брахмагупти-Фібоначчі: ( a 2 + b 2 ) ( c 2 + d 2 ) = = ( a c - b d ) 2 + ( a d + b c ) 2

Рівняння виду xy = k (гіпербола)

Задача представляє собою розв'язання діофантового рівняння вигляду x y = k , де k - задане ціле відмінне від нуля число.

Таке рівняння має геометричну інтерпретацію. На координатній площині воно описує гіперболу, асимптотами якої є осі координат. Знаходження цілих чисел x та y геометрично означає пошук точок, які знаходяться чітко на перетині гілок цієї гіперболи та ліній цілочисельної сітки площини.

Розв'язання цього рівняння зводиться до знаходження всіх дільників числа k . Оскільки добуток двох цілих невідомих дорівнює константі, то кожна змінна повинна бути цілочисельним дільником цієї константи. На відміну від кола, гіпербола завжди має скінченну кількість розв'язків.

Для знаходження всіх можливих розв'язків використовується наступний алгоритм:
Спочатку знаходиться множина всіх дільників заданого числа k , включаючи як додатні, так і від'ємні дільники.
Далі для кожного знайденого дільника формується пара розв'язків:

  • Значення першої змінної прирівнюється до знайденого дільника: x = d .
  • Значення другої змінної обчислюється шляхом ділення константи на цей дільник: y = k d .
  • Якщо число k є додатним, знаки дільників у парі будуть однаковими (обидва плюси або обидва мінуси). Якщо k від'ємне, то знаки в парі будуть різними.

Таким чином, загальна кількість цілочисельних розв'язків (точок на гіперболі) завжди дорівнює кількості всіх цілих дільників числа k .
